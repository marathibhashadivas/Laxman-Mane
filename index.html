लक्ष्मण माने

(१ जून १९४९ - )

भारतीय सामाजिक कार्यकर्ते व विख्यात लेखक लक्ष्मण माने “उपरा” कार या नावाने ही प्रसिद्ध आहेत. मागासवर्गीय, भटक्या विमुक्तांच्या प्रश्नांचा सखोल अभ्यास असलेल्या लक्ष्मण माने यांच्या लेखनात “समाजातील उपेक्षित” विषयावरील परिवर्तनवादी व पुरोगामी विचार व्यक्त होतात.

महत्वाचे साहित्यिक योगदान:

  • उपरा ( आत्मकथन )
  • उध्वस्त
  • क्रांतिपथ ( कविता संग्रह )
  • खेळ साडेतीन टक्क्याचा ( लेख संग्रह )
  • प्रकाशपुत्र ( नाटक )

पुरस्कार:

  • न. चि. केळकर पुरस्कार
  • पद्मश्री
  • फोर्ड फाउंडेशन पुरस्कार
  • भरती विद्यापीठ पुरस्कार
  • साहित्य अकादमी पुरस्कार
